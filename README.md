# Examiner – AI Powered Mock Interview Platform

Examiner is an AI-powered mock interview platform designed to simulate real-world technical and HR interviews. The system generates personalized interview questions based on either a candidate's uploaded resume or selected subject area, creating a dynamic and adaptive interview experience.

The platform features multiple AI interviewers represented through interactive 3D avatars with voice synthesis and lip-sync animations, providing an immersive interview environment. Candidates can respond through text or voice, while the system evaluates performance and generates detailed interview reports with technical, communication, and confidence scores.

Key Features:

* Resume-based interview generation using PDF text extraction
* Subject-based interviews (Java, React, Python, DBMS, DSA, etc.)
* AI-generated adaptive questions with Easy, Moderate, and Hard difficulty levels
* Interactive 3D interviewer avatars with speech and facial animations
* Browser-based Text-to-Speech for interviewer voices
* Voice-to-Text response transcription
* Automated interview evaluation and performance analytics
* Dynamic final reports with strengths, weaknesses, and improvement suggestions

Tech Stack:

* Frontend: React.js, Vite, Tailwind CSS, Three.js, React Three Fiber
* AI Models: Groq LLM API, Whisper Speech Recognition
* Backend: FastAPI (Python)
* 3D Technologies: VRM Avatars, Three.js
* State Management: React Context API

Examiner aims to help students and job seekers improve interview skills, build confidence, and prepare effectively for placement drives and professional interviews.
